## О игре

**Robo-Dismantle: Neon Protocol** — это браузерный экшн-платформер, в котором вы управляете модульным роботом, сражающимся с волнами вражеских дронов. Главная особенность игры — **система повреждений конечностей**: враги могут целиться и отстреливать ваши конечности по одной, что напрямую влияет на игровую механику.

---

## Уникальные особенности

### Система конечностей

Ваш робот состоит из **6 модульных конечностей**, каждая со своим запасом прочности:

| Конечность | HP | Функция |
|------------|-----|---------|
| Голова | 50 | Потеря не фатальна, но уменьшает защиту |
| Торс | 100 | **Вital** — уничтожение = конец игры |
| Левая рука | 50 | Стрельба левой кнопкой мыши |
| Правая рука | 50 | Стрельба правой кнопкой мыши |
| Левая нога | 75 | Скорость перемещения и прыжки |
| Правая нога | 75 | Скорость перемещения и прыжки |

**Эффекты потери конечностей:**
- Потеря **1 ноги** — скорость движения снижается на 50%
- Потеря **2 ног** — скорость снижается на 80%, ползание
- Потеря **руки** — невозможно использовать соответствующее оружие
- Каждая активная конечность даёт **15% сопротивления урону** для жизненно важных компонентов

### Боевая система

- **Двуручное оружие**: ЛКМ (Плазма — голубой), ПКМ (Лазер — жёлтый)
- **Перезарядка**: Разная скорострельность для каждого оружия
- **Атаки сверху**: Прыжок на врага наносит урон
- **Оглушение**: При получении урона — временный стан и отбрасывание

### ИИ врагов

- Дроны поддерживают дистанцию от игрока
- Стратегическая стрельба с прицеливанием в конечности
- Динамическая частота появления, зависящая от счёта
- Спавн с разных сторон (слева, справа, сверху)

## Управление

| Клавиша | Действие |
|---------|----------|
| `A` / `D` | Движение влево / вправо |
| `W` / `Space` | Прыжок |
| `S` | Присесть |
| `ЛКМ` | Плазменный выстрел |
| `ПКМ` | Лазерный выстрел |
| `R` | Рестарт после поражения |

## Установка и запуск

### Предварительные требования

- **Node.js** (версия 18 или выше)

### Шаги

1. **Клонирование репозитория**
   ```bash
   git clone <repository-url>
   cd robo-dismantle_-neon-protocol
   ```

2. **Установка зависимостей**
   ```bash
   npm install
   ```

3. **Запуск приложения**
   ```bash
   npm run dev
   ```

4. **Откройте в браузере**
   ```
   http://localhost:5173
   ```

## Игровые механики

### Физика
- Гравитация: 0.6 единиц/кадр²
- Трение: 0.8 (сопротивление воздуха)
- Высота прыжка: -14 единиц

### Баланс сложности
- Частота появления врагов увеличивается каждые 2000 очков
- Урон зависит от типа конечности
- Стан-механика для тактического геймплея

## Особенности реализации
